[Api] Api Servers of industry4.0 and Auto Guided Vehicle

To avoid the direct connection between client and databases, and let components take their own role, so we construct the api server.

Construct environment : NodeJS, ExpressJS, Sequelize, momentJS, Lodash, Promise

§ AGV Api Server

Create Date: 2017/5

Create a restful Api server to connect databases for website and data-processing.

In this repo, we classify it into two parts, Agv and WMS(Warehouse Management System), also website is.

§ Robot Api Server

Create Date: 2017/3

For realize the production line’s realtime data and static data on website, we also need a api server to be a communicational bridge.

The website part is inroduced in article named NKG Industry4.0 management page, but backend and api part have nothing document or screenshot to show because of the confidential content.
